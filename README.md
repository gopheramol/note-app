# note-app
A Simple and Efficient Web-Based Note-taking App in Go.

## Stack
### Go
### Mongo

# Installation
Make sure Docker and Docker Compose installed

## Run the application

```docker-compose up```

## Browse localhost

``` http://localhost:8081/ ```
