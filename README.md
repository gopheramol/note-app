# note-app
A Simple and Efficient Web-Based Note-taking App in Go.

## Stack
- Go
- MongoDB

# Installation
Make sure Docker and Docker Compose are installed.

## Run the application
```bash
docker-compose up
``` 
## Browse localhost
Navigate to http://localhost:8081/ in your web browser.
